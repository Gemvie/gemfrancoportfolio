# Gemvie Franco — Portfolio

Personal portfolio eCommerce Data Specialist based in Cebu, Philippines.

Built with React, TypeScript, Vite, and Tailwind CSS.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
